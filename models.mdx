FlexInference proxies **any** OpenAI model for `default`, `priority`, and `auto` requests. The models below additionally support the **flex race** - the duration form of [`start_within`](/deadline-routing). Pass the model by its **alias** (for example `gpt-5.5`), not a dated snapshot.

A **duration** `start_within` on a model that isn't on this list returns `400 model_not_flex_capable` - the flex race only works on flex-capable models. Those same models still work with `default`, `priority`, or `auto`, which proxy any model straight to OpenAI.

## Inputs

These models accept the input types you'd expect from OpenAI:

- **Text** — system, developer, user, and assistant messages.
- **Images** — pass image URLs or base64 data URLs. Data URLs are the most reliable.
- **Files** — PDFs and other file inputs supported by the Responses API.

Output is text (including tool calls, structured outputs, and reasoning). Streaming works on every model.

<Note>
  New flex-capable models are added as OpenAI ships them. If a model you expect is missing, check that you're using its alias and that OpenAI offers flex pricing for it.
</Note>
